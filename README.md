# Landing page for Fortify

#### Prepare
- Clone latest version this repository
```bash
npm i
```

#### Run development
```bash
npm run development
```
Visit [http://localhost:3000](http://localhost:3000)

#### Run build
```bash
npm run build
```

#### Run deploy
```bash
npm run deploy
```
Visit [https://fortifyapp.com](https://fortifyapp.com)
